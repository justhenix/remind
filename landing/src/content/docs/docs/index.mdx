**Linters catch syntax. remindy catches taste**: AI-slop copy, bespoke UI, vague commits, security foot-guns, features nobody asked for.

Better models won't fix it: they still don't know how _you_ work, and forget it when you switch tools. Teach remindy once; it holds every agent to your standards before it writes.

<CardGrid stagger>
  <Card title="Portable via MCP" icon="puzzle">
    A correction made in one tool (say Cursor) is respected by another (Kiro,
    Claude, Windsurf). One shared memory, every agent.
  </Card>
  <Card title="About 15 tokens a turn" icon="rocket">
    Each rule compresses to a single caveman line, so recall costs almost nothing
    per code write.
  </Card>
  <Card title="Runs on your machine" icon="laptop">
